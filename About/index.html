Trigram (C++)

This program uses trigram frequencies to identify the natural language in which a document is written. The program takes in several documents of known and unknown languages. For an unknown language, the program compares its contents to documents of known languages. The result of the comparisons to each known language are sorted and outputting as a result.

Six Degree (C++)

The purpose of this program is to store information about teaching assistants and courses. It store a list of classes and students, class rosters, and the teaching assistants for each class. It can output a list of students, a list of courses, a class roster, the classes a student assisted in, and the path from one student to another.
